# Gestion-residuos
La aplicacion fue desarrollada con "FLASK" para registrar puntos de reciclaje.
Se permite Ingresar Nombre, Tipo de residuo como también su Ubicación, Mostrando los registros en una tabla de manera sencilla.

-INSTALACION-
Copiar en el repositorio:
git clone https://github.com/willymaldonado98-afk/Gestion-residuos.git
   cd Gestion-residuos

Instalar dependecias en el terminal : pip install flask

-EJECUCION-

en app.py en el terminal iniciamos la aplicación: python app.py

luego abrir en el navegador: http://127.0.0.1:5000

Estructura del Proyecto
